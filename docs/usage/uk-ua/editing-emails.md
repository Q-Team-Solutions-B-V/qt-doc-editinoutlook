# Редагування електронних листів

На цій сторінці описано, як використовувати Edit in Outlook для редагування та надсилання електронних листів для документів Business Central через Microsoft Outlook.

## Основний робочий процес

### Покрокова процедура

#### 1. Вибір документа
1. **Відкрийте документ Business Central** (наприклад, комерційна пропозиція, рахунок-фактура, замовлення)
2. **Натисніть «Надіслати електронний лист»** на стрічці
3. **Відкриється діалогове вікно електронного листа**

#### 2. Активація Edit in Outlook
1. **У діалоговому вікні електронного листа**:
   - Виберіть одержувачів
   - Виберіть шаблон електронного листа (необов'язково)
   - **Натисніть «Редагувати в Outlook»**
2. **Браузер починає завантаження** файлу .eml
3. **Зачекайте завершення завантаження**

#### 3. Редагування в Outlook
1. **Відкрийте завантажений файл .eml** (подвійне клацання)
2. **Outlook відкривається** із попередньо заповненим електронним листом:
   - Документ як вкладення PDF
   - Вміст шаблону в тілі листа
   - Інформацію про одержувача заповнено
3. **Відредагуйте електронний лист** за потреби:
   - Налаштуйте тему
   - Додайте особистий текст
   - Додайте додаткові вкладення
   - Налаштуйте форматування

#### 4. Надсилання
1. **Перегляньте електронний лист**
2. **Натисніть «Надіслати»** в Outlook
3. **Електронний лист надсилається** через ваш особистий обліковий запис Outlook
4. **Електронний лист з'являється** у папці Outlook «Надіслані елементи»

## Функції редагування електронних листів

### Коригування тексту
В Outlook ви можете:
- **Змінити тему**
- **Відкоригувати текст повідомлення**:
  - Додати особисту нотатку
  - Змінити текст шаблону
  - Налаштувати форматування (жирний, курсив, кольори)
  - Додати списки
  - Вставити посилання

### Керування вкладеннями
- **Переглядати наявні вкладення** (документ PDF)
- **Додавати додаткові вкладення**:
  - Додаткові документи
  - Зображення
  - Специфікації
  - Контракти
- **Видаляти вкладення** (за потреби)
- **Перейменовувати вкладення**

### Налаштування одержувачів
- **Розширити поле «Кому»** додатковими одержувачами
- **Додати одержувачів у копію**
- **Прихована копія для невидимих копій**
- **Перевіряти адреси** через адресну книгу Outlook

### Підпис і форматування
- **Особистий підпис** додається автоматично
- **Корпоративний стиль** збережено із шаблону
- **Форматування HTML** повністю доступне
- **Редагування форматованого тексту** за допомогою редактора Outlook

## Розширені функції

### Персоналізація шаблону
Налаштовуйте шаблони відповідно до ситуації:

```html
<!-- Перед надсиланням -->
<p>Шановний(-а) %CUSTOMER_NAME%,</p>
<p>У додатку надсилаємо %DOCUMENT_TYPE% %DOCUMENT_NO%.</p>

<!-- Після редагування в Outlook -->
<p>Шановний пане Коваленко,</p>
<p>Відповідно до нашої сьогоднішньої розмови, у додатку надсилаємо
переглянуту комерційну пропозицію КП001 з оновленими цінами.</p>
```

### Масове редагування електронних листів
Для кількох документів:
1. **Використайте Edit in Outlook** для першого документа
2. **Збережіть як шаблон** в Outlook
3. **Застосуйте шаблон** до наступних документів
4. **Персоналізуйте** кожне повідомлення
