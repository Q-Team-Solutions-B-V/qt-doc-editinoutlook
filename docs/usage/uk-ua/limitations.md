# Обмеження

На цій сторінці описано відомі обмеження, обмеження та міркування під час використання Edit in Outlook.

## Технічні обмеження

### Обмеження браузера та завантаження

#### Обмеження розміру файлу
| Компонент | Максимальний розмір | Причина | Вирішення |
|-----------|--------------------|---------|-----------| 
| **Файл EML** | 25 МБ | Ліміт завантаження браузера | Розділіть великі вкладення |
| **Вкладення PDF** | 20 МБ | Ліміти поштового клієнта | Стисніть PDF |
| **Весь електронний лист** | 30 МБ | Ліміт Outlook/Exchange | Використовуйте посилання SharePoint |
| **Control Add-in** | 100 МБ пам'яті | Продуктивність браузера | Оновіть сторінку |

#### Проблеми, специфічні для браузерів
- **Chrome**: Блокувальник спливаючих вікон може блокувати завантаження
- **Firefox**: Проблеми з кодуванням символів, що не входять до ASCII
- **Edge**: Іноді виникає перевищення часу очікування з великими файлами
- **Safari**: Обмежена підтримка попереднього перегляду PDF

### Обмеження платформи Business Central

#### SaaS та локальна установка
| Функціональність | SaaS | Локальна | Примітка |
|------------------|------|----------|---------|
| **Автоматичні оновлення** | ✅ | ❌ | SaaS автоматично, локально вручну |
| **Настроювані розширення** | ⚠️ Обмежено | ✅ Повністю | Обмеження пісочниці SaaS |
| **Режим налагодження** | ❌ | ✅ | Політика безпеки SaaS |
| **Доступ до журналів** | ⚠️ Application Insights | ✅ Прямий | Різні підходи до ведення журналів |

#### Ліміти API
- **Запитів на хвилину**: 100 (SaaS), Необмежено (локально)
- **Тайм-аут сеансу**: 30 хвилин за замовчуванням
- **Одночасні користувачі**: Відповідно до моделі ліцензування

## Обмеження Outlook та Office

### Outlook Desktop та веб-версія

#### Порівняння функцій
| Функція | Desktop | Веб | Мобільний |
|---------|---------|-----|---------- |
| **Відкриття EML** | ✅ Вбудоване | ⚠️ Через браузер | ❌ Обмежено |
| **Розширене форматування** | ✅ Повністю | ✅ Повністю | ⚠️ Базове |
| **Обробка вкладень** | ✅ Повністю | ✅ Повністю | ⚠️ Тільки завантаження |
| **Доступ без мережі** | ✅ Повністю | ❌ Відсутній | ⚠️ Обмежено |

#### Залежності від версій
- **Outlook 2016/2019**: Обмежена підтримка EML
- **OWA Legacy**: Без прямого відкриття EML
- **Мобільні додатки**: Потребує підготовчого кроку на комп'ютері

## Обмеження типів документів

### Підтримуваний та непідтримуваний

#### Повністю підтримується
- ✅ Комерційні пропозиції, замовлення, рахунки-фактури продажу
- ✅ Замовлення, рахунки-фактури закупівлі
- ✅ Проведені документи
- ✅ Кредит-ноти
- ✅ Складські відвантаження

#### Обмежена підтримка
- ⚠️ **Сервісні документи**: Базова підтримка шаблонів
- ⚠️ **Проектні документи**: Мінімальна персоналізація
- ⚠️ **Замовлення на складання**: Немає специфічних шаблонів

#### Не підтримується
- ❌ **Об'єкти настроюваних звітів**: Немає генерації EML
- ❌ **Інтеграція з Dataverse**: Тільки власні функції BC
- ❌ **Зовнішні документи**: Немає підтримки третіх сторін

### Обмеження шаблонів

#### Проблеми шаблонів HTML

```html
/* Проблематичний CSS */
.nepidtrymuvanyi {
    position: absolute;  /* Не широко підтримується */
    transform: rotate(); /* Проблеми поштового клієнта */
    /* @media queries — Обмежена підтримка */
}
```

#### Підтримуваний підмножина HTML
- ✅ Базові теги HTML (p, div, table, br)
- ✅ Стилі CSS (перевага вбудованим)
- ✅ Зображення (вбудовані або пов'язані)
- ❌ JavaScript (заблоковано з міркувань безпеки)
- ❌ Складні макети CSS (flexbox, grid)
- ❌ Завантаження зовнішніх шрифтів

## Обмеження користувачів та прав

### Залежності дозволів

#### Мінімально необхідні права
Користувачу потрібно мати щонайменше:
- **Читання документів**: Для доступу до документів
- **Читання шаблону електронного листа**: Для доступу до шаблонів
- **Завантаження файлів**: Для файлів EML
- **Виконання Control Add-in**: Для функціональності браузера
